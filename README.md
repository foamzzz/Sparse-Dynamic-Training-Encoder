# Sparse-Dynamic-Training-Encoder
The code and data repository for the paper `Sparse Dynamic Training Encoder: A Novel Approach for Enhancing Cross-Subject Generalization in SSVEP-BCI`.

The dataset originates from the Brain-Computer Interface Research Group at Tsinghua University. Recently, the original website appears to be under maintenance and is currently inaccessible.
Original URL: `https://bci.med.tsinghua.edu.cn/download.html`

I am providing the link to the copy I previously downloaded here, strictly for academic and research purposes only.
